# 微视频录制指南

## 基本信息

- **微视频主题**：在希望的田野上——农村改革与中国的春天
- **选定歌曲**：《在希望的田野上》（彭丽媛原唱）
- **视频时长**：约9分钟（6-10分钟范围之内）
- **表演形式**：5人分段落独唱 + 5人口述历史分析

---

## 视频结构

| 时间段 | 内容 | 时长 |
|--------|------|------|
| 0:00-0:10 | 片头：显示微视频主题 | 10秒 |
| 0:10-3:40 | 完整演唱（5人各一个镜头，间奏播农村风光） | 3分30秒 |
| 3:40-8:40 | 口述分析：5人依次出镜 | 5分钟 |
| 8:40-9:00 | 片尾：班级+5人姓名 | 20秒 |

## 录制步骤

### 第一步：下载伴奏

从 QQ 音乐/网易云搜索"在希望的田野上 原版伴奏"，下载后发到群里。**所有人必须用同一个文件**。

### 第二步：每个人分别录制演唱视频

**详细见 [歌词分配与录制方案.md](./歌词分配与录制方案.md)**：

- 吴思琪：第一段（家乡）
- 蹇琬颐：第一段续（田野丰收+副歌）
- 罗应杰：第二段（理想）
- 王思懿：第二段续（四方劳动+副歌）
- 彭熙桐：第三段（未来，全段+副歌）

**录制要点**：
- 戴耳机听伴奏唱，手机横屏录自己
- 干净背景（白墙），半身像，看镜头
- 从头到尾完整唱一遍（约3分钟，不要只录自己段落）
- 开始后先安静等3秒再开口

### 第三步：录制口述部分

每人对着 [视频口述脚本.md](./视频口述脚本.md) 逐字稿朗读：
- 吴思琪：开场
- 蹇琬颐：小岗村18个红手印
- 罗应杰：家庭联产承包责任制
- 王思懿：词曲作家采风故事
- 彭熙桐：结尾

### 第四步：后期制作（剪映）

- [ ] 片头：黑底标题
- [ ] 演唱部分：5人镜头依次拼接，间奏放农村照片
- [ ] 口述部分：5人依次出镜
- [ ] 背景音乐：伴奏降音量垫底
- [ ] 片尾：班级+5人姓名
- [ ] 全程字幕（剪映自动识别+校对）

### 第五步：导出

1920x1080 / 30fps / MP4 / 500MB以内
