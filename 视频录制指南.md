# 微视频录制指南

## 基本信息

- **微视频主题**：春天的故事——改革开放的历史回响
- **选定歌曲**：《春天的故事》（董文华原唱版本）
- **视频时长**：6-10分钟
- **表演形式**：小组合唱 + 口述历史分析

---

## 视频结构

| 时间段 | 内容 | 时长 |
|--------|------|------|
| 0:00-0:10 | 片头：显示微视频主题 | 10秒 |
| 0:10-6:10 | 完整演唱《春天的故事》两段（6人各一个镜头，间奏播今昔对比） | 6分钟 |
| 6:10-9:10 | 口述分析：6人依次出镜，每人30秒 | 3分钟 |
| 8:30-9:00 | 片尾：显示班级和全体成员名字 | 30秒 |

**总时长约 9 分钟。**

## 录制步骤

### 第一步：下载伴奏（所有人统一用同一个文件）

从 QQ 音乐/网易云搜索"春天的故事 原版伴奏"，下载后发到 6 人群里。**所有人必须用同一个伴奏文件**，否则后期对不齐。

### 第二步：每个人分别录制演唱视频

**详细方案见 [歌词分配与录制方案.md](./歌词分配与录制方案.md)**

核心要点：
- 戴耳机听伴奏，用手机横屏录自己（这样只录到你的清唱声，无伴奏残留）
- 选干净背景（白墙最好），半身像，看镜头
- 从头到尾完整唱一遍（不要只唱分配的部分），约2分30秒
- 开始录制后先安静等 3 秒再开口（给剪辑留对齐点）

### 第三步：录制口述分析部分

按 [歌词分配与录制方案.md](./歌词分配与录制方案.md) 中的口述分配：
- **罗应杰** 讲开场 + 1979年部分
- **吴思琪** 讲深圳奇迹 + 1992年部分
- **叶显平** 讲结尾升华

逐字稿见 [视频口述脚本.md](./视频口述脚本.md)

### 第四步：后期制作（可由一人统一完成）

**推荐工具**：剪映（手机/电脑均可）

**制作清单**：
- [ ] 片头：黑底白色标题"春天的故事——改革开放的历史回响"，10秒
- [ ] 演唱部分：按歌词分配表切换画面，合唱部分用画中画 3x2 同框
- [ ] 口述部分：穿插历史图片（深圳今昔对比、邓小平视察照片等，搜索引擎可找到公共资源）
- [ ] 全程加背景音乐（伴奏降音量垫底）
- [ ] 片尾：黑底白字，班级和6人姓名
- [ ] 全程字幕（剪映自动识别 + 手动校对）

### 第五步：导出

- 1920x1080 / 30fps / MP4 / 500MB 以内

## 历史图片素材获取

讲述部分可以配合以下画面（搜索引擎搜索关键词即可找到公共资源）：

1. **1979年深圳旧貌** — 搜索"深圳 1979 老照片"
2. **蛇口开山第一炮** — 搜索"蛇口工业区 1979"
3. **邓小平视察深圳** — 搜索"邓小平 深圳 1984 1992"
4. **深圳今昔对比** — 搜索"深圳 改革开放 今昔对比"
5. **邓小平南方谈话** — 搜索"邓小平 南方谈话 1992"
6. **深圳现代天际线** — 搜索"深圳 天际线 2023"
